Fitness:- Schedule of Fitness

A responsive fitness-themed website designed to provide users with tools and resources to maintain a healthy lifestyle. This website includes interactive features like a BMI calculator, calorie calculator, fitness exercises, and a contact page.

Features


Homepage: Engaging layout showcasing the gym/fitness theme.

BMI Calculator: Allows users to calculate their Body Mass Index based on height and weight.

Calorie Calculator: Helps users estimate their daily calorie needs.

Fitness Exercises: Provides a list of exercises for different muscle groups and fitness goals.

Contact Page: Lets users get in touch with the website owner or gym.


Technologies Used

HTML5 for the structure of web pages.

CSS3 for styling and responsiveness.

JavaScript for interactive functionality (BMI and Calorie calculators).


Responsive Design

The website is fully responsive and optimized for desktop, tablet, and mobile devices, ensuring a smooth user 

experience across all screen sizes.
